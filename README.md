パイロット版プロトタイプ
=============

## フォルダ構成

```
.
├── index.html
├── *.html
│
├── /css /* スタイルフォルダ */
│   ├── *.css
│   ├── *.less
│   └── *.scss
│
├── /img /* 画像フォルダ */
│   └── *.png, *.jpg, *.gif
│
└── /js /* JSフォルダ */
    ├── *.js
    ├── /tmpl /* テンプレートフォルダ */
    │    └── *.js
    └── /lib /* ライブラリ系JSフォルダ */
    　    └── *.js
```

## 使用ライブラリ

* jQuery
* Underscore.js
* moment.js
